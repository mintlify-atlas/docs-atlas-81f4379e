MaRCoRe's AI Protocol Hub is a living collection of protocols, behavioral contracts, and small tools built from real use — not theory. Every project here started as a solution to a specific problem: AI sessions drifting off course, decisions disappearing between chats, context collapsing under its own weight. These tools are designed to be used alongside an AI assistant, not just read.
